# 🚀 Космический шутер

Аркадная игра для Telegram Mini App. Управляйте НЛО, сбивайте астероиды и инопланетян!

## Управление
- Мышь/палец — движение
- Клик/тап — стрельба

## Особенности
- 3 жизни
- Звуки
- Пауза по клавише P
- Звёздная анимация

## Стек
HTML5 Canvas, JavaScript, Telegram Mini App API
